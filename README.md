Họ tên: Đinh Trà My

## Mục tiêu

Hoàn thành Phiếu 01 — tách data.php / index.php, in bảng 8 SKU, map tên danh mục, tính đúng tổng giá trị kho, escape HTML và thêm MS_EXPECT.

## Cấu trúc repo

- `data.php` — chứa dữ liệu danh mục & 8 sản phẩm MiniShop
- `index.php` — hiển thị bảng catalog, tính thành tiền, tổng giá trị kho, debug

## Cách chạy

1. Mở XAMPP Control Panel → Start **Apache**
2. Đặt folder `cse485-cm-01` vào `htdocs/`
3. Truy cập: `http://localhost/cse485-cm-01/`
